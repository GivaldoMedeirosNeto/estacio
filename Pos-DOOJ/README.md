# ☕ Desenvolvimento Orientado a Objetos com Java

Repositório contendo exercícios, projetos e aplicações desenvolvidas durante os estudos de Desenvolvimento Orientado a Objetos com Java.

Os projetos abrangem desde conceitos básicos da linguagem até aplicações Web utilizando Java EE.

---

# 🚀 Objetivo

O objetivo deste repositório é armazenar práticas, exercícios e aplicações desenvolvidas durante as disciplinas de:

- Programação Básica Java JEE
- Programação Avançada Java JSE
- Aplicação Web Java EE

---

# 🛠️ Tecnologias Utilizadas

- Java
- Java SE
- Java EE
- JSP / Servlet
- JavaFX
- Threads
- Collections
- Orientação a Objetos
- HTML5
- CSS3
